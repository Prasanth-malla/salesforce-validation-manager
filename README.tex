# Salesforce Validation Rule Manager

## Features
- Salesforce OAuth Login
- Fetch Validation Rules
- Show Active/Inactive State
- Toggle Validation Rules
- Update Rules Directly in Salesforce

## Tech Stack
- React.js
- Node.js
- Express.js
- Salesforce Tooling API
- OAuth 2.0 PKCE

## Run Backend
cd backend
node server.js

## Run Frontend
cd frontend
npm start
